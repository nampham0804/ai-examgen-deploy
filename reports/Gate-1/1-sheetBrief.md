# 1-PAGE BRIEF: AI QUIZ MATRIX GENERATOR

| Mục | Nội dung |
| :--- | :--- |
| **Problem**<br>*(Vấn đề)* | • Giảng viên mất quá nhiều thời gian và công sức để tạo đề thi trắc nghiệm thủ công trên Moodle. Các hệ thống hiện tại thường xuyên xảy ra lỗi:<br>• Lỗi định dạng nghiêm trọng khi câu hỏi có chứa mã nguồn (code) hoặc hình ảnh (ví dụ: toán tử `=` trong code bị hệ thống nhận diện nhầm thành ký hiệu đáp án đúng của file GIFT dẫn đến lỗi logic đề).<br>• Quy trình chọn câu hỏi thủ công phức tạp, không lưu trạng thái đang làm, nếu mất tập trung là phải xóa hết làm lại từ đầu. |
| **Target User**<br>*(Người dùng)* | • Giảng viên (đặc biệt trong khối ngành Kỹ thuật/CNTT) sử dụng LMS Moodle để tổ chức thi hoặc kiểm tra đánh giá. |
| **Solution**<br>*(Giải pháp)* | • Ứng dụng AI tự động đọc hiểu tài liệu thô để sinh ngân hàng câu hỏi chuẩn hóa, kết hợp với bộ cấu hình ma trận thông minh giúp tự động bốc đề ngẫu nhiên và đóng gói file xuất bản (GIFT/XML) không lỗi code/hình ảnh. |
| **Core Features**<br>*(Tính năng chính)* | • **AI Document Parser:** Upload tài liệu thô (Đề cương, slide, giáo trình) $\rightarrow$ AI tự động phân tích và sinh câu hỏi trắc nghiệm chia theo Bài học và 3 Mức độ.<br>• **Smart Code-Block Wrapper:** Tự động phát hiện và bọc các đoạn code/hình ảnh vào đúng định dạng hiển thị, xử lý triệt để xung đột ký tự đặc biệt khi import vào Moodle.<br>• **Matrix Random Picker:** Người dùng chỉ cần nhập số lượng câu hỏi theo ma trận bài học/mức độ, hệ thống tự động bốc ngẫu nhiên ra các bộ đề khác nhau (tránh lỗi đề rỗng).<br>• **Moodle-Ready Exporter:** Xuất file định dạng GIFT/XML/PDF/DOC/DOCX chuẩn Moodle chỉ với 1 cú click. |
| **Value Proposition**<br>*(Giá trị mang lại)* | • **Tiết kiệm 90% thời gian:** Thay vì copy-paste và định dạng thủ công từng câu hỏi/mã nguồn, giảng viên chỉ cần nạp tài liệu và nhập ma trận đề thi.<br>• **Chính xác:** Loại bỏ hoàn toàn các lỗi hiển thị code, lỗi trùng đáp án do xung đột cú pháp, và lỗi đề rỗng. |
| **Success Metrics**<br>*(Đo lường)* | • Thời gian trung bình để tạo và export một bộ đề thi 50 câu (chứa code/hình ảnh) giảm xuống dưới 3 phút.<br>• Tỷ lệ file GIFT/XML import thành công vào Moodle ngay từ lần đầu tiên đạt $> 98\%$ (không bị lỗi format hay nhận diện sai đáp án). |
| **Tech Stack**<br>*(Công nghệ)* | • **Frontend:** Next.js, Tailwind CSS (Xây dựng bảng cấu hình ma trận trực quan).<br>• **Backend:** FastAPI (Python) - tối ưu cho việc xử lý file và tích hợp các thư viện phân tích dữ liệu.<br>• **AI Integration:** Claude API / Gemini API (Sử dụng các dòng mô hình có khả năng phân tích mã nguồn tốt để sinh câu hỏi và xử lý định dạng). |
